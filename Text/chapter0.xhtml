The first line has no indent because of the class "noIndent". If you want to indent it, simply remove the class (class="noIndent"). If, on the other hand, you wish to put all your text without indents go to styles.css and remove the following bold line: 

      p {
        margin: 0;
        text-align: justify;
        text-indent: 1.5em;
        line-height: 1.25em;
      }

If you wish to put the text in bold you can use "<b>...</b>", to make it italic use "<i>...</i>", and to underline it use "<u>...</u>". You can also add interactive footnotes 1 that, when you click on it, it takes you to the explanation bellow.
